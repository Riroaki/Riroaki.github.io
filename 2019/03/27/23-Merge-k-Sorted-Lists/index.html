23.Merge k Sorted Lists

Posted on 2019-03-27 | Edited on 2019-03-28 | In leetcode

题目

Merge k sorted linked lists and return it as one sorted list. Analyze and describe its complexity.

Example:

题解

将全部元素加入list，排序后转回链表

  • 时间复杂度：$O(nlog(n))$，其中n是总节点数
  • 空间复杂度：$O(n)$

使用最小堆（优先队列）

  • 时间复杂度：$O(nlogk)$
  • 空间复杂度：$O(n+k)​$

一个接一个合并

  • 时间复杂度：$O(kn)​$
  • 空间复杂度：$O(n)$

分治法，两两合并，然后再合并……

  • 这是实践效果最快的！
  • 时间复杂度：$O(nlogk)​$
  • 空间复杂度：$O(n+log(k))$，其中$log(k)$为递归的栈深度
